# Guarantees App (نظام إدارة الضمانات)

نظام متكامل لإدارة الضمانات البنكية والنقدية، يدعم واجهة ويب (Flask) وتطبيق سطح مكتب (PyQt5).

## المميزات
- **لوحة تحكم (Dashboard):** عرض إحصائيات الضمانات (النشطة، المنتهية، المعلقة).
- **إدارة البيانات:** دعم لقواعد البيانات SQLite و PostgreSQL.
- **تقارير:** تصدير تقارير Excel و PDF.
- **تسجيل الدخول:** دعم تسجيل الدخول عبر Google OAuth.
- **واجهة مستخدم:** واجهة عصرية تدعم اللغة العربية (RTL).

## متطلبات التشغيل (Deployment)
المشروع جاهز للرفع على منصات مثل Render أو Heroku.
- `Procfile`: معد لتشغيل التطبيق باستخدام Gunicorn.
- `requirements.txt`: يحتوي على جميع المكتبات اللازمة.
- `runtime.txt`: يحدد إصدار Python 3.13.1.

## التشغيل المحلي (Local Development)
1. تثبيت المتطلبات:
   ```bash
   pip install -r requirements.txt
   ```
2. تشغيل تطبيق الويب:
   ```bash
   python web_app.py
   ```
   أو عبر السكربت: `run_web_app.bat`

## الملاحظات
- لتفعيل دخول Google، راجع ملف `INSTRUCTIONS.txt` لإعداد مفاتيح API.
- التطبيق يدعم العمل الهجين (سطح مكتب + ويب).
